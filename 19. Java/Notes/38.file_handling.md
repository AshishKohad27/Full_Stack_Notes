# 38. File Handling

- Java can read, write, create, and delete files
- Main classes: `File`, `FileWriter`, `FileReader`, `BufferedReader`, `BufferedWriter`
- All in `java.io` package

## 1. Create & Check File

```java
import java.io.File;

class Hello {
    public static void main(String args[]) {

        File file = new File("test.txt");

        System.out.println("Exists: " + file.exists());      // true/false
        System.out.println("Name: " + file.getName());        // test.txt
        System.out.println("Path: " + file.getAbsolutePath());
        System.out.println("Is File: " + file.isFile());      // true
        System.out.println("Is Dir: " + file.isDirectory());   // false
    }
}
```

## 2. Write to File

```java
import java.io.FileWriter;
import java.io.IOException;

class Hello {
    public static void main(String args[]) {

        try {
            // Write (creates file if not exists, OVERWRITES if exists)
            FileWriter writer = new FileWriter("notes.txt");
            writer.write("Hello from Java!\n");
            writer.write("This is line 2\n");
            writer.close();
            System.out.println("File written!");

            // Append (add to existing content)
            FileWriter appender = new FileWriter("notes.txt", true); // true = append mode
            appender.write("This line is appended\n");
            appender.close();
            System.out.println("Content appended!");

        } catch (IOException e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}
```

## 3. Read from File

```java
import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

class Hello {
    public static void main(String args[]) {

        try {
            BufferedReader reader = new BufferedReader(new FileReader("notes.txt"));
            String line;

            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }

            reader.close();
        } catch (IOException e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}
// Output:
// Hello from Java!
// This is line 2
// This line is appended
```

## 4. Delete File

```java
import java.io.File;

class Hello {
    public static void main(String args[]) {

        File file = new File("notes.txt");

        if (file.delete()) {
            System.out.println("Deleted: " + file.getName());
        } else {
            System.out.println("Failed to delete");
        }
    }
}
```

## 5. Try-with-Resources (Auto-close) -- Best Practice

```java
import java.io.*;

class Hello {
    public static void main(String args[]) {

        // File automatically closed after try block - no need for .close()
        try (BufferedReader reader = new BufferedReader(new FileReader("notes.txt"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            System.out.println("Error: " + e.getMessage());
        }
        // reader is automatically closed here!
    }
}
```

## Interview Points
- Always close files (use try-with-resources)
- `FileWriter(file, true)` = append mode
- `BufferedReader` is faster than `FileReader` (reads chunks, not char by char)
- `try-with-resources` (Java 7+) automatically closes resources
