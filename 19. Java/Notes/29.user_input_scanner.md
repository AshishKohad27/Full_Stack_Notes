# User Input - Scanner

- Scanner class is used to **take input from user** (keyboard)
- Must import: `import java.util.Scanner;`

## 1. Basic Input

```java
import java.util.Scanner;

class Hello {
    public static void main(String args[]) {

        // Create Scanner object to read input
        Scanner sc = new Scanner(System.in);

        // Take integer input
        System.out.print("Enter your age: ");
        int age = sc.nextInt();
        System.out.println("Your age is: " + age);

        // Take double input
        System.out.print("Enter price: ");
        double price = sc.nextDouble();
        System.out.println("Price is: " + price);

        // Take string input (single word)
        System.out.print("Enter your name: ");
        String name = sc.next(); // reads only ONE word
        System.out.println("Hello " + name);

        // Take full line input (multiple words)
        sc.nextLine(); // clear the buffer (important after nextInt/next)
        System.out.print("Enter your address: ");
        String address = sc.nextLine(); // reads FULL line
        System.out.println("Address: " + address);

        // Take boolean input
        System.out.print("Are you student (true/false): ");
        boolean isStudent = sc.nextBoolean();
        System.out.println("Student: " + isStudent);

        sc.close(); // always close scanner when done
    }
}
```

## 2. Example - Simple Calculator with User Input

```java
import java.util.Scanner;

class Hello {
    public static void main(String args[]) {

        Scanner sc = new Scanner(System.in);

        System.out.print("Enter first number: ");
        int num1 = sc.nextInt();

        System.out.print("Enter second number: ");
        int num2 = sc.nextInt();

        int sum = num1 + num2;
        int diff = num1 - num2;
        int product = num1 * num2;

        System.out.println("Sum: " + sum);
        System.out.println("Difference: " + diff);
        System.out.println("Product: " + product);

        if (num2 != 0)
            System.out.println("Division: " + (num1 / num2));
        else
            System.out.println("Cannot divide by zero!");

        sc.close();
    }
}
```

## 3. Example - Grade Checker

```java
import java.util.Scanner;

class Hello {
    public static void main(String args[]) {

        Scanner sc = new Scanner(System.in);

        System.out.print("Enter your marks (0-100): ");
        int marks = sc.nextInt();

        if (marks >= 90)
            System.out.println("Grade: A+");
        else if (marks >= 80)
            System.out.println("Grade: A");
        else if (marks >= 70)
            System.out.println("Grade: B");
        else if (marks >= 60)
            System.out.println("Grade: C");
        else if (marks >= 50)
            System.out.println("Grade: D");
        else
            System.out.println("Grade: F (Fail)");

        sc.close();
    }
}
```

## Scanner Methods Summary
| Method           | Reads               | Example Input |
|------------------|----------------------|---------------|
| `nextInt()`      | Integer              | 42            |
| `nextDouble()`   | Decimal number       | 3.14          |
| `nextFloat()`    | Float number         | 5.5f          |
| `nextLong()`     | Long number          | 123456789L    |
| `next()`         | Single word (String) | Hello         |
| `nextLine()`     | Full line (String)   | Hello World   |
| `nextBoolean()`  | true or false        | true          |
