# 1. What is Java?

- Java is a **programming language** created by **James Gosling** in 1995 at Sun Microsystems
- Java is **platform independent** - write once, run anywhere (Windows, Mac, Linux)
- Java is **Object-Oriented** - everything is inside a class

## How Java Works?

```
You write code (.java file)
        |
        v
Compiler (javac) converts to Bytecode (.class file)
        |
        v
JVM (Java Virtual Machine) runs the Bytecode
        |
        v
Output on screen
```

## Important Terms

| Term | Full Form                  | What it does                              |
|------|----------------------------|-------------------------------------------|
| JDK  | Java Development Kit       | Tools to write + compile + run Java       |
| JRE  | Java Runtime Environment   | Tools to run Java programs only           |
| JVM  | Java Virtual Machine       | Reads bytecode and runs it on your machine|

- **JDK = JRE + Compiler + Tools**
- **JRE = JVM + Libraries**
- You need JDK to develop, JRE to just run

## Your First Java Program

```java
class Hello {
    public static void main(String args[]) {
        System.out.println("Hello World!");
    }
}
```

### How to run:
```
Step 1: Save file as Hello.java (file name MUST match class name)
Step 2: Open terminal
Step 3: javac Hello.java    (compile - creates Hello.class)
Step 4: java Hello           (run - shows output)
```

### Breaking down the code:
```
class Hello                  --> Every Java program needs a class
public                       --> Accessible from anywhere
static                       --> Can run without creating object
void                         --> Returns nothing
main                         --> Entry point - program starts here
String args[]                --> Command line arguments (ignore for now)
System.out.println()         --> Prints text and goes to next line
```
