# 10. Strings

- String = sequence of characters
- Strings are **immutable** (cannot be changed after creation) -- INTERVIEW FAVORITE!

## 1. Creating Strings

```java
class Hello {
    public static void main(String args[]) {

        // Way 1: String literal (stored in String Pool)
        String s1 = "Hello";

        // Way 2: Using new keyword (stored in Heap)
        String s2 = new String("Hello");

        // Both store "Hello" but in DIFFERENT memory locations
    }
}
```

## 2. Important String Methods

```java
class Hello {
    public static void main(String args[]) {

        String name = "Ashish Kumar";

        // Length
        System.out.println(name.length());            // 12

        // Character at index
        System.out.println(name.charAt(0));            // A
        System.out.println(name.charAt(6));            // space

        // Substring
        System.out.println(name.substring(7));         // Kumar
        System.out.println(name.substring(0, 6));      // Ashish

        // Uppercase / Lowercase
        System.out.println(name.toUpperCase());        // ASHISH KUMAR
        System.out.println(name.toLowerCase());        // ashish kumar

        // Trim (remove spaces from start & end)
        String spaced = "  Hello  ";
        System.out.println(spaced.trim());             // "Hello"

        // Contains
        System.out.println(name.contains("Kumar"));    // true

        // Starts with / Ends with
        System.out.println(name.startsWith("Ash"));    // true
        System.out.println(name.endsWith("mar"));      // true

        // Replace
        System.out.println(name.replace("Kumar", "K")); // Ashish K

        // Split
        String csv = "Apple,Banana,Mango";
        String fruits[] = csv.split(",");
        for (String f : fruits)
            System.out.println(f);  // Apple  Banana  Mango

        // Index of
        System.out.println(name.indexOf("Kumar"));     // 7
        System.out.println(name.indexOf("xyz"));       // -1 (not found)

        // Is empty
        String empty = "";
        System.out.println(empty.isEmpty());           // true
    }
}
```

## 3. String Comparison -- INTERVIEW IMPORTANT!

```java
class Hello {
    public static void main(String args[]) {

        String s1 = "Hello";
        String s2 = "Hello";
        String s3 = new String("Hello");

        // == checks REFERENCE (memory address), NOT value
        System.out.println(s1 == s2);       // true  (same pool reference)
        System.out.println(s1 == s3);       // false (different memory!)

        // .equals() checks VALUE -- ALWAYS USE THIS!
        System.out.println(s1.equals(s3));            // true
        System.out.println(s1.equalsIgnoreCase("HELLO")); // true
    }
}
```

## 4. String Immutability -- WHY IT MATTERS

```java
class Hello {
    public static void main(String args[]) {

        String s = "Hello";
        s.concat(" World"); // creates NEW string, original NOT changed
        System.out.println(s); // Hello (still same!)

        // To get the new value, you must REASSIGN
        s = s.concat(" World");
        System.out.println(s); // Hello World

        // Every modification creates a NEW string object
        // This wastes memory if you modify strings a lot
        // Solution: use StringBuilder or StringBuffer
    }
}
```

## 5. StringBuilder (Mutable, Fast, NOT thread-safe)

```java
class Hello {
    public static void main(String args[]) {

        StringBuilder sb = new StringBuilder("Hello");

        sb.append(" World");          // adds at end
        System.out.println(sb);       // Hello World

        sb.insert(5, " Java");        // insert at index 5
        System.out.println(sb);       // Hello Java World

        sb.replace(6, 10, "Beautiful");
        System.out.println(sb);       // Hello Beautiful World

        sb.delete(5, 16);             // delete from index 5 to 15
        System.out.println(sb);       // Hello World

        sb.reverse();
        System.out.println(sb);       // dlroW olleH

        // Convert back to String
        String result = sb.toString();
    }
}
```

## 6. StringBuffer (Same as StringBuilder but thread-safe, slower)

```java
class Hello {
    public static void main(String args[]) {

        StringBuffer sb = new StringBuffer("Hello");
        sb.append(" World");
        System.out.println(sb); // Hello World
        // Same methods as StringBuilder
    }
}
```

## Interview Quick Reference
| Feature        | String          | StringBuilder   | StringBuffer    |
|----------------|-----------------|-----------------|-----------------|
| Mutable?       | No (immutable)  | Yes             | Yes             |
| Thread-safe?   | Yes (immutable) | No              | Yes             |
| Speed           | Slow (creates new obj) | Fast     | Medium          |
| When to use?   | Few changes     | Many changes, single thread | Many changes, multi thread |

## String Pool -- INTERVIEW CONCEPT
- When you write `String s = "Hello"`, Java checks String Pool first
- If "Hello" already exists in pool, it **reuses** it (saves memory)
- `new String("Hello")` always creates **new object** in heap (ignores pool)
