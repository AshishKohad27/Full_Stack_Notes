# 6. Operators

> ref: `Code/5.ArithmeticOperators/`

## A. Arithmetic Operators

```java
class Hello {
    public static void main(String args[]) {

        int a = 20;
        int b = 7;

        System.out.println("Add: " + (a + b));       // 27
        System.out.println("Subtract: " + (a - b));   // 13
        System.out.println("Multiply: " + (a * b));    // 140
        System.out.println("Divide: " + (a / b));      // 2 (quotient, not decimal)
        System.out.println("Modulus: " + (a % b));     // 6 (remainder)

        // Shorthand operators
        int x = 10;
        x += 5;   // x = x + 5  --> 15
        x -= 3;   // x = x - 3  --> 12
        x *= 2;   // x = x * 2  --> 24
        x /= 4;   // x = x / 4  --> 6
        x %= 4;   // x = x % 4  --> 2

        // Increment & Decrement
        int num = 5;
        num++;            // num = 6 (post increment)
        ++num;            // num = 7 (pre increment)
        num--;            // num = 6 (post decrement)

        // Pre vs Post - IMPORTANT DIFFERENCE
        int val = 5;
        int pre = ++val;  // first increment, then assign --> pre=6, val=6
        System.out.println("Pre: " + pre + ", val: " + val);

        val = 5;
        int post = val++; // first assign, then increment --> post=5, val=6
        System.out.println("Post: " + post + ", val: " + val);
    }
}
```

## B. Relational Operators (Compare two values, returns boolean)

```java
class Hello {
    public static void main(String args[]) {

        int x = 10;
        int y = 20;

        System.out.println(x < y);    // true  (less than)
        System.out.println(x > y);    // false (greater than)
        System.out.println(x <= y);   // true  (less than or equal)
        System.out.println(x >= y);   // false (greater than or equal)
        System.out.println(x == y);   // false (equal to)
        System.out.println(x != y);   // true  (not equal to)
    }
}
```

## C. Logical Operators (Combine conditions)

```java
class Hello {
    public static void main(String args[]) {

        int age = 25;
        boolean hasLicense = true;

        // AND (&&) - BOTH must be true
        boolean canDrive = (age >= 18) && hasLicense;
        System.out.println("Can drive: " + canDrive); // true

        // OR (||) - ANY ONE must be true
        boolean hasDiscount = (age < 12) || (age > 60);
        System.out.println("Has discount: " + hasDiscount); // false

        // NOT (!) - flips true to false
        boolean isMinor = !(age >= 18);
        System.out.println("Is minor: " + isMinor); // false
    }
}
```

### Short-circuit behavior:
- `&&` --> if first condition is **false**, second is **not checked**
- `||` --> if first condition is **true**, second is **not checked**
