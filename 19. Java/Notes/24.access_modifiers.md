# Access Modifiers

> ref: `Code/24.packages/`

- Access Modifiers control **who can access** your variables, methods, and classes
- Think of it like a door lock - you decide who can enter

## 4 Types of Access Modifiers

| Modifier    | Same Class | Same Package | Child Class (other pkg) | Other Package |
|-------------|------------|--------------|-------------------------|---------------|
| `public`    | Yes        | Yes          | Yes                     | Yes           |
| `protected` | Yes        | Yes          | Yes                     | No            |
| `default`   | Yes        | Yes          | No                      | No            |
| `private`   | Yes        | No           | No                      | No            |

## Example

```java
class Student
{
    public String name;       // anyone can access
    protected int age;        // same package + child classes can access
    int marks;                // default - only same package can access (no keyword needed)
    private String password;  // ONLY this class can access

    // To access private variable from outside, use getter/setter
    public String getPassword()
    {
        return password;
    }

    public void setPassword(String password)
    {
        this.password = password;
    }
}

class Hello {
    public static void main(String args[]) {

        Student s1 = new Student();

        s1.name = "Ashish";         // public - works everywhere
        s1.age = 22;                // protected - works (same package)
        s1.marks = 90;              // default - works (same package)
        // s1.password = "123";     // ERROR! private - cannot access directly

        s1.setPassword("123");      // use setter to set private variable
        System.out.println(s1.getPassword()); // use getter to get private variable

        System.out.println(s1.name);   // Ashish
        System.out.println(s1.age);    // 22
        System.out.println(s1.marks);  // 90
    }
}
```

## Simple Way to Remember
- `private` = only ME (my class)
- `default` = me + my NEIGHBORS (same package)
- `protected` = me + neighbors + my CHILDREN (child classes)
- `public` = EVERYONE
