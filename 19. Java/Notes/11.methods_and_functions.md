# 11. Methods (Functions)

> ref: `Code/12.MethodOverloading/`

- Method = block of code that performs a task
- Reusable - write once, call many times
- Structure: `accessModifier returnType methodName(parameters) { body }`

## 1. Method with No Return, No Parameters

```java
class Hello {

    // Method definition
    static void greet()
    {
        System.out.println("Hello, Welcome!");
    }

    public static void main(String args[]) {

        greet(); // Method call
        greet(); // Call again - reusable!
        // Output:
        // Hello, Welcome!
        // Hello, Welcome!
    }
}
```

## 2. Method with Parameters

```java
class Hello {

    static void greet(String name, int age)
    {
        System.out.println("Hello " + name + ", you are " + age);
    }

    public static void main(String args[]) {

        greet("Ashish", 22);  // Hello Ashish, you are 22
        greet("Rahul", 25);   // Hello Rahul, you are 25
    }
}
```

## 3. Method with Return Value

```java
class Hello {

    static int add(int a, int b)
    {
        return a + b; // returns value back to caller
    }

    static boolean isEven(int num)
    {
        return num % 2 == 0;
    }

    public static void main(String args[]) {

        int sum = add(10, 20);
        System.out.println("Sum: " + sum); // Sum: 30

        System.out.println(isEven(4)); // true
        System.out.println(isEven(7)); // false
    }
}
```

## 4. Method in a Class (Non-static)

```java
class Calculator {
    int add(int a, int b) { return a + b; }
    int sub(int a, int b) { return a - b; }
}

class Hello {
    public static void main(String args[]) {

        // Need to create OBJECT to call non-static method
        Calculator calc = new Calculator();
        System.out.println(calc.add(5, 3)); // 8
        System.out.println(calc.sub(5, 3)); // 2
    }
}
```

## 5. Method Overloading (Same name, different parameters)

```java
class Hello {

    static int add(int a, int b)       { return a + b; }
    static int add(int a, int b, int c){ return a + b + c; }
    static double add(double a, double b) { return a + b; }

    public static void main(String args[]) {

        System.out.println(add(5, 3));       // 8 (calls int version)
        System.out.println(add(5, 3, 2));    // 10 (calls 3-param version)
        System.out.println(add(1.5, 2.5));   // 4.0 (calls double version)

        // Java decides WHICH method to call based on arguments
        // This is COMPILE-TIME polymorphism
    }
}
```

## Interview Points
- `static` method: call directly without object (`Hello.add()`)
- Non-static method: need object to call (`calc.add()`)
- Method Overloading: same name, different parameter list
- Return type ALONE cannot differentiate overloaded methods
- Java is **Pass by Value** (copies are passed, not original variable)
