# 8. Loops

> ref: `Code/9.Loops/`

- Loops = repeat a block of code **multiple times**
- 3 types: `while`, `do-while`, `for`

## 1. while loop (check FIRST, then run)

```java
class Hello {
    public static void main(String args[]) {

        int i = 1;

        while (i <= 5) // condition checked FIRST
        {
            System.out.println("Hello " + i);
            i++;
        }
        // Output:
        // Hello 1
        // Hello 2
        // Hello 3
        // Hello 4
        // Hello 5

        // If condition is false from start, loop NEVER runs
        int x = 10;
        while (x < 5) // false from start
        {
            System.out.println("This won't print");
        }
    }
}
```

## 2. do-while loop (run FIRST, then check)

```java
class Hello {
    public static void main(String args[]) {

        int i = 1;

        do
        {
            System.out.println("Count: " + i);
            i++;
        } while (i <= 5); // condition checked AFTER

        // Output:
        // Count: 1
        // Count: 2
        // Count: 3
        // Count: 4
        // Count: 5

        // KEY DIFFERENCE: runs AT LEAST ONCE even if condition is false
        int x = 10;
        do
        {
            System.out.println("Runs once! x=" + x); // THIS WILL PRINT
            x++;
        } while (x < 5); // false, but code already ran once

        // Output: Runs once! x=10
    }
}
```

## 3. for loop (most common)

```java
class Hello {
    public static void main(String args[]) {

        //  for (start; condition; increment)
        for (int i = 1; i <= 5; i++)
        {
            System.out.println("Number: " + i);
        }
        // Output:
        // Number: 1
        // Number: 2
        // Number: 3
        // Number: 4
        // Number: 5

        // Reverse loop
        for (int i = 5; i >= 1; i--)
        {
            System.out.print(i + " "); // 5 4 3 2 1
        }
    }
}
```

## 4. Nested Loop (loop inside loop)

```java
class Hello {
    public static void main(String args[]) {

        // Print a pattern:
        // *
        // * *
        // * * *
        for (int i = 1; i <= 3; i++)        // rows
        {
            for (int j = 1; j <= i; j++)    // columns
            {
                System.out.print("* ");
            }
            System.out.println(); // new line after each row
        }
    }
}
```

## 5. break and continue

```java
class Hello {
    public static void main(String args[]) {

        // break - EXIT the loop completely
        for (int i = 1; i <= 10; i++)
        {
            if (i == 5)
                break; // stop at 5
            System.out.print(i + " "); // 1 2 3 4
        }

        System.out.println();

        // continue - SKIP current iteration, go to next
        for (int i = 1; i <= 5; i++)
        {
            if (i == 3)
                continue; // skip 3
            System.out.print(i + " "); // 1 2 4 5
        }
    }
}
```

## When to use which?
- **for** --> when you know HOW MANY times to loop
- **while** --> when you DON'T know how many times (loop until condition)
- **do-while** --> when you want to run AT LEAST ONCE
