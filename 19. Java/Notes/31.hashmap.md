# 31. HashMap (Key-Value Pairs)

- HashMap = stores data as **key-value** pairs (like a dictionary)
- Keys must be **unique**, values can repeat
- Part of `java.util` package
- **Unordered** (no guaranteed order)

## 1. Basic HashMap

```java
import java.util.HashMap;

class Hello {
    public static void main(String args[]) {

        // HashMap<KeyType, ValueType>
        HashMap<String, Integer> marks = new HashMap<>();

        // PUT - add key-value pair
        marks.put("Ashish", 90);
        marks.put("Rahul", 85);
        marks.put("Priya", 95);
        System.out.println(marks); // {Ashish=90, Priya=95, Rahul=85}

        // GET - get value by key
        System.out.println(marks.get("Ashish")); // 90
        System.out.println(marks.get("xyz"));     // null (key not found)

        // SIZE
        System.out.println(marks.size()); // 3

        // CHECK if key/value exists
        System.out.println(marks.containsKey("Rahul"));    // true
        System.out.println(marks.containsValue(95));        // true

        // REMOVE
        marks.remove("Rahul");
        System.out.println(marks); // {Ashish=90, Priya=95}

        // UPDATE (put with existing key replaces value)
        marks.put("Ashish", 99);
        System.out.println(marks.get("Ashish")); // 99

        // GET with default value
        System.out.println(marks.getOrDefault("xyz", 0)); // 0 (key not found, returns default)
    }
}
```

## 2. Loop Through HashMap

```java
import java.util.HashMap;
import java.util.Map;

class Hello {
    public static void main(String args[]) {

        HashMap<String, Integer> marks = new HashMap<>();
        marks.put("Ashish", 90);
        marks.put("Rahul", 85);
        marks.put("Priya", 95);

        // Way 1: Loop through entries (MOST COMMON)
        for (Map.Entry<String, Integer> entry : marks.entrySet()) {
            System.out.println(entry.getKey() + " = " + entry.getValue());
        }

        // Way 2: Loop through keys only
        for (String name : marks.keySet()) {
            System.out.println(name + " : " + marks.get(name));
        }

        // Way 3: Loop through values only
        for (int mark : marks.values()) {
            System.out.println("Mark: " + mark);
        }
    }
}
```

## 3. Real World Example - Word Frequency Counter

```java
import java.util.HashMap;

class Hello {
    public static void main(String args[]) {

        String sentence = "apple banana apple mango banana apple";
        String words[] = sentence.split(" ");

        HashMap<String, Integer> freq = new HashMap<>();

        for (String word : words) {
            if (freq.containsKey(word)) {
                freq.put(word, freq.get(word) + 1); // increment count
            } else {
                freq.put(word, 1); // first occurrence
            }
        }

        System.out.println(freq); // {banana=2, apple=3, mango=1}
    }
}
```

## HashMap vs Other Maps
| Map Type          | Ordered? | Sorted? | Thread-safe? |
|-------------------|----------|---------|--------------|
| `HashMap`         | No       | No      | No           |
| `LinkedHashMap`   | Yes (insertion order) | No | No   |
| `TreeMap`         | Yes      | Yes (by key) | No      |
| `Hashtable`       | No       | No      | Yes          |

## Interview Points
- Key must be unique, value can duplicate
- `null` key is allowed (only one), `null` values are allowed (multiple)
- Default initial capacity: 16, load factor: 0.75
- Time complexity: `get()`, `put()` = O(1) average
- Uses `hashCode()` and `equals()` to store and find keys
