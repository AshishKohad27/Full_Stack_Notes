# 16. this Keyword

- `this` = reference to the **current object**
- Used to distinguish between instance variable and parameter with same name

## 1. this with Variables

```java
class Student {
    String name;
    int age;

    Student(String name, int age) {
        // Without this: name = name (parameter assigns to itself, NOT to instance variable!)
        this.name = name; // this.name = instance variable, name = parameter
        this.age = age;
    }

    void display() {
        System.out.println(this.name + " : " + this.age);
        // "this" is optional here since there's no ambiguity
    }
}

class Hello {
    public static void main(String args[]) {

        Student s = new Student("Ashish", 22);
        s.display(); // Ashish : 22
    }
}
```

## 2. this() to Call Another Constructor

```java
class Student {
    String name;
    int age;

    Student() {
        this("Unknown", 0); // calls the parameterized constructor below
        System.out.println("Default constructor");
    }

    Student(String name, int age) {
        this.name = name;
        this.age = age;
        System.out.println("Parameterized constructor");
    }
}

class Hello {
    public static void main(String args[]) {

        Student s = new Student(); // calls default, which calls parameterized
        // Output:
        // Parameterized constructor
        // Default constructor

        System.out.println(s.name); // Unknown
    }
}
```

## 3. this to Pass Current Object

```java
class Printer {
    void print(Student s) {
        System.out.println("Printing: " + s.name);
    }
}

class Student {
    String name;

    Student(String name) {
        this.name = name;
    }

    void printMe() {
        Printer p = new Printer();
        p.print(this); // passing CURRENT object to another method
    }
}

class Hello {
    public static void main(String args[]) {

        Student s = new Student("Ashish");
        s.printMe(); // Printing: Ashish
    }
}
```

## Interview Points
- `this` cannot be used in static methods (static has no object)
- `this()` must be the FIRST statement in constructor
- `this` refers to current object, `super` refers to parent object
