# 3. Variables & Data Types

> ref: `Code/2.Variables/`

- Variable = a **container** to store data
- Data Type = **what kind** of data the variable holds

## Primitive Data Types (8 types)

```java
class Hello {
    public static void main(String args[]) {

        // ===== 1. INTEGER TYPES =====
        byte b = 127;           // -128 to 127            (1 byte)
        short s = 32000;        // -32768 to 32767        (2 bytes)
        int num = 100;          // -2 billion to 2 billion (4 bytes) -- MOST USED
        long l = 99999L;        // very large numbers      (8 bytes) -- put L at end

        // ===== 2. DECIMAL TYPES =====
        float f = 5.8f;         // decimal numbers         (4 bytes) -- put f at end
        double d = 5.8;         // decimal numbers         (8 bytes) -- MOST USED

        // ===== 3. CHARACTER TYPE =====
        char c = 'A';           // single character        (2 bytes) -- use single quotes

        // ===== 4. BOOLEAN TYPE =====
        boolean isTrue = true;  // true or false           (1 bit)

        System.out.println("byte: " + b);       // 127
        System.out.println("short: " + s);      // 32000
        System.out.println("int: " + num);      // 100
        System.out.println("long: " + l);       // 99999
        System.out.println("float: " + f);      // 5.8
        System.out.println("double: " + d);     // 5.8
        System.out.println("char: " + c);       // A
        System.out.println("boolean: " + isTrue); // true
    }
}
```

## Non-Primitive Data Types
```java
class Hello {
    public static void main(String args[]) {

        // String - sequence of characters (use double quotes)
        String name = "Ashish";

        // Array - collection of same type
        int numbers[] = {10, 20, 30};

        System.out.println(name);       // Ashish
        System.out.println(numbers[0]); // 10
    }
}
```

## Variable Rules
- Must start with letter, `_` or `$` (not number)
- Cannot use Java keywords (int, class, void etc.)
- Java is **case sensitive** - `name` and `Name` are different
- Use **camelCase** for variables: `myFirstName`, `totalMarks`
