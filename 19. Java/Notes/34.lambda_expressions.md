# 34. Lambda Expressions (Java 8+)

- Lambda = **short way to write a method** (anonymous function)
- Used with **functional interfaces** (interface with only ONE abstract method)
- Syntax: `(parameters) -> { body }`

## 1. Without Lambda vs With Lambda

```java
// Functional Interface (has only ONE abstract method)
interface Greeting {
    void sayHello(String name);
}

class Hello {
    public static void main(String args[]) {

        // WITHOUT Lambda - old way (anonymous class)
        Greeting g1 = new Greeting() {
            public void sayHello(String name) {
                System.out.println("Hello " + name);
            }
        };
        g1.sayHello("Ashish"); // Hello Ashish

        // WITH Lambda - short and clean!
        Greeting g2 = (name) -> {
            System.out.println("Hello " + name);
        };
        g2.sayHello("Rahul"); // Hello Rahul

        // Even shorter (single statement = no braces needed)
        Greeting g3 = name -> System.out.println("Hello " + name);
        g3.sayHello("Priya"); // Hello Priya
    }
}
```

## 2. Lambda with Return Value

```java
interface Calculator {
    int calculate(int a, int b);
}

class Hello {
    public static void main(String args[]) {

        // Add
        Calculator add = (a, b) -> a + b;
        System.out.println(add.calculate(5, 3)); // 8

        // Subtract
        Calculator sub = (a, b) -> a - b;
        System.out.println(sub.calculate(10, 4)); // 6

        // Multiply
        Calculator mul = (a, b) -> a * b;
        System.out.println(mul.calculate(3, 7)); // 21

        // Same interface, different behavior! That's the power of lambda
    }
}
```

## 3. Lambda with forEach (Most Common Usage)

```java
import java.util.ArrayList;
import java.util.Arrays;

class Hello {
    public static void main(String args[]) {

        ArrayList<String> names = new ArrayList<>(Arrays.asList("Ashish", "Rahul", "Priya"));

        // Old way
        for (String name : names) {
            System.out.println(name);
        }

        // Lambda way
        names.forEach(name -> System.out.println(name));

        // Even shorter - method reference
        names.forEach(System.out::println);

        // Lambda with numbers
        ArrayList<Integer> nums = new ArrayList<>(Arrays.asList(1, 2, 3, 4, 5));
        nums.forEach(n -> System.out.println(n * n)); // 1 4 9 16 25
    }
}
```

## 4. Lambda with Sorting

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.Collections;

class Hello {
    public static void main(String args[]) {

        ArrayList<String> names = new ArrayList<>(Arrays.asList("Charlie", "Alice", "Bob"));

        // Sort alphabetically
        Collections.sort(names, (a, b) -> a.compareTo(b));
        System.out.println(names); // [Alice, Bob, Charlie]

        // Sort by length
        Collections.sort(names, (a, b) -> a.length() - b.length());
        System.out.println(names); // [Bob, Alice, Charlie]

        // Sort in reverse
        Collections.sort(names, (a, b) -> b.compareTo(a));
        System.out.println(names); // [Charlie, Bob, Alice]
    }
}
```

## Lambda Syntax Shortcuts
```java
// Full form
(int a, int b) -> { return a + b; }

// Types can be removed (Java infers them)
(a, b) -> { return a + b; }

// Single expression = no braces, no return keyword
(a, b) -> a + b

// Single parameter = no parentheses
name -> System.out.println(name)

// No parameter
() -> System.out.println("Hello")
```

## Interview Points
- Lambda works only with **Functional Interface** (1 abstract method)
- `@FunctionalInterface` annotation ensures interface has only 1 method
- Lambda is NOT a method, it's an **expression** (instance of functional interface)
- Method Reference: `System.out::println` is shortcut for `x -> System.out.println(x)`
