# 4. Literals

- Literal = the **actual value** assigned to a variable

```java
class Hello {
    public static void main(String args[]) {

        // Normal integer
        int num = 100;

        // Underscore in numbers (for readability, Java ignores underscores)
        int crore = 10_00_00_000;    // 10 crore = 100000000
        System.out.println(crore);    // 100000000

        // Binary literal (starts with 0b)
        int binary = 0b1010;         // binary of 10
        System.out.println(binary);   // 10

        // Hexadecimal literal (starts with 0x)
        int hex = 0xFF;              // hex of 255
        System.out.println(hex);      // 255

        // Octal literal (starts with 0)
        int octal = 017;             // octal of 15
        System.out.println(octal);    // 15

        // Scientific notation (e = x10^)
        double sci = 12e3;           // 12 x 10^3 = 12000.0
        System.out.println(sci);      // 12000.0

        // Character literal (char stores ASCII number)
        char ch = 'A';
        ch++;                         // A(65) + 1 = B(66)
        System.out.println(ch);       // B

        // boolean literal - only true or false
        boolean b = true;
        // boolean b2 = 1; // ERROR! Java boolean is NOT 0/1 like C
    }
}
```
