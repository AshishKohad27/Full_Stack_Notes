# 15. Static Keyword

- `static` = belongs to the **CLASS**, not to any specific object
- Shared by ALL objects of that class

## 1. Static Variable

```java
class Student {
    String name;              // different for each object
    static String school = "ABC School"; // SHARED by all objects

    Student(String name) {
        this.name = name;
    }

    void display() {
        System.out.println(name + " - " + school);
    }
}

class Hello {
    public static void main(String args[]) {

        Student s1 = new Student("Ashish");
        Student s2 = new Student("Rahul");

        s1.display(); // Ashish - ABC School
        s2.display(); // Rahul - ABC School

        // Change school for ALL students at once
        Student.school = "XYZ School"; // access using CLASS name

        s1.display(); // Ashish - XYZ School
        s2.display(); // Rahul - XYZ School
        // Both changed! Because static is SHARED
    }
}
```

## 2. Static Method

```java
class MathHelper {

    // Static method - call WITHOUT creating object
    static int add(int a, int b) {
        return a + b;
    }

    static int square(int n) {
        return n * n;
    }
}

class Hello {
    public static void main(String args[]) {

        // No need to create object!
        int sum = MathHelper.add(5, 3);
        int sq = MathHelper.square(4);

        System.out.println("Sum: " + sum); // 8
        System.out.println("Square: " + sq); // 16
    }
}
```

## 3. Static Block (Runs ONCE when class is loaded)

```java
class Config {

    static String dbUrl;
    static int maxRetries;

    // Static block - runs ONCE before main, before any object creation
    static {
        System.out.println("Static block executed!");
        dbUrl = "localhost:3306";
        maxRetries = 3;
    }
}

class Hello {
    public static void main(String args[]) {

        System.out.println(Config.dbUrl);       // localhost:3306
        System.out.println(Config.maxRetries);   // 3
        // "Static block executed!" printed BEFORE these lines
    }
}
```

## 4. Static vs Non-Static Rules

```java
class Demo {
    int x = 10;         // non-static (belongs to object)
    static int y = 20;  // static (belongs to class)

    static void staticMethod() {
        // System.out.println(x); // ERROR! Cannot access non-static from static
        System.out.println(y);    // OK! static can access static
    }

    void normalMethod() {
        System.out.println(x);    // OK! non-static can access non-static
        System.out.println(y);    // OK! non-static can access static too
    }
}
```

## Quick Rules
| Rule                                        | Allowed? |
|---------------------------------------------|----------|
| Static method accesses static variable      | Yes      |
| Static method accesses non-static variable  | No       |
| Non-static method accesses static variable  | Yes      |
| Non-static method accesses non-static       | Yes      |
| Call static method without object            | Yes      |
| Call non-static method without object        | No       |

## Interview Points
- `static` = class level, shared by all objects
- `main()` is static because JVM calls it without creating an object
- Static methods cannot use `this` or `super`
- Static block runs only ONCE when class loads
