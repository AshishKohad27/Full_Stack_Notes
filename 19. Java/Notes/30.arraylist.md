# 30. ArrayList (Dynamic Array)

- ArrayList = **resizable array** (size grows/shrinks automatically)
- Part of `java.util` package
- Stores **objects only** (use Wrapper classes for primitives)

## 1. Basic ArrayList

```java
import java.util.ArrayList;

class Hello {
    public static void main(String args[]) {

        // Create ArrayList of Strings
        ArrayList<String> names = new ArrayList<>();

        // Add elements
        names.add("Ashish");
        names.add("Rahul");
        names.add("Priya");
        System.out.println(names); // [Ashish, Rahul, Priya]

        // Access element (by index)
        System.out.println(names.get(0)); // Ashish
        System.out.println(names.get(2)); // Priya

        // Size
        System.out.println(names.size()); // 3

        // Update element
        names.set(1, "Raj");
        System.out.println(names); // [Ashish, Raj, Priya]

        // Remove element
        names.remove(1);           // by index
        names.remove("Priya");     // by value
        System.out.println(names); // [Ashish]

        // Check if exists
        System.out.println(names.contains("Ashish")); // true

        // Clear all
        names.clear();
        System.out.println(names.isEmpty()); // true
    }
}
```

## 2. ArrayList with Numbers

```java
import java.util.ArrayList;

class Hello {
    public static void main(String args[]) {

        // Use Integer (wrapper) not int
        ArrayList<Integer> nums = new ArrayList<>();
        nums.add(10);  // autoboxing: int --> Integer
        nums.add(20);
        nums.add(30);
        nums.add(40);

        // Loop - for loop
        for (int i = 0; i < nums.size(); i++) {
            System.out.print(nums.get(i) + " "); // 10 20 30 40
        }

        System.out.println();

        // Loop - for-each (simpler)
        for (int n : nums) {
            System.out.print(n + " "); // 10 20 30 40
        }
    }
}
```

## 3. ArrayList of Objects

```java
import java.util.ArrayList;

class Student {
    String name;
    int marks;

    Student(String name, int marks) {
        this.name = name;
        this.marks = marks;
    }

    public String toString() {
        return name + ":" + marks;
    }
}

class Hello {
    public static void main(String args[]) {

        ArrayList<Student> students = new ArrayList<>();
        students.add(new Student("Ashish", 90));
        students.add(new Student("Rahul", 85));
        students.add(new Student("Priya", 95));

        for (Student s : students) {
            System.out.println(s.name + " scored " + s.marks);
        }
        // Ashish scored 90
        // Rahul scored 85
        // Priya scored 95

        System.out.println(students); // [Ashish:90, Rahul:85, Priya:95]
    }
}
```

## 4. Sorting ArrayList

```java
import java.util.ArrayList;
import java.util.Collections;

class Hello {
    public static void main(String args[]) {

        ArrayList<Integer> nums = new ArrayList<>();
        nums.add(40);
        nums.add(10);
        nums.add(30);
        nums.add(20);

        Collections.sort(nums);           // ascending
        System.out.println(nums);          // [10, 20, 30, 40]

        Collections.reverse(nums);         // reverse
        System.out.println(nums);          // [40, 30, 20, 10]

        System.out.println(Collections.min(nums)); // 10
        System.out.println(Collections.max(nums)); // 40
    }
}
```

## Array vs ArrayList
| Feature      | Array               | ArrayList                |
|--------------|----------------------|--------------------------|
| Size         | Fixed                | Dynamic (grows/shrinks)  |
| Type         | Primitive + Objects  | Objects only (wrappers)  |
| Performance  | Faster               | Slightly slower          |
| Methods      | Only `.length`       | add, remove, get, set... |
| Syntax       | `int[] a = new int[5]` | `ArrayList<Integer> a = new ArrayList<>()` |
