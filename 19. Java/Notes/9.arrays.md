# 9. Arrays

- Array = collection of **same type** elements stored in **continuous memory**
- Array size is **fixed** - once created, cannot change size
- Index starts from **0**

## 1. Creating Arrays

```java
class Hello {
    public static void main(String args[]) {

        // Way 1: Declare size, then assign values
        int nums[] = new int[5]; // creates array of size 5 (default values: 0)
        nums[0] = 10;
        nums[1] = 20;
        nums[2] = 30;
        nums[3] = 40;
        nums[4] = 50;

        // Way 2: Declare with values directly
        int marks[] = {90, 85, 70, 60, 95};

        // Access element
        System.out.println(marks[0]); // 90 (first element)
        System.out.println(marks[4]); // 95 (last element)

        // Array length
        System.out.println(marks.length); // 5
    }
}
```

## 2. Loop Through Array

```java
class Hello {
    public static void main(String args[]) {

        int marks[] = {90, 85, 70, 60, 95};

        // for loop
        for (int i = 0; i < marks.length; i++)
        {
            System.out.println("Index " + i + " = " + marks[i]);
        }

        // for-each loop (simpler, no index needed)
        for (int m : marks)
        {
            System.out.println("Mark: " + m);
        }
    }
}
```

## 3. 2D Array (Table / Matrix)

```java
class Hello {
    public static void main(String args[]) {

        // 2D Array = array of arrays (rows & columns)
        int matrix[][] = {
            {1, 2, 3},    // row 0
            {4, 5, 6},    // row 1
            {7, 8, 9}     // row 2
        };

        // Access: matrix[row][column]
        System.out.println(matrix[0][0]); // 1
        System.out.println(matrix[1][2]); // 6
        System.out.println(matrix[2][1]); // 8

        // Print entire 2D array
        for (int i = 0; i < matrix.length; i++)
        {
            for (int j = 0; j < matrix[i].length; j++)
            {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println();
        }
        // Output:
        // 1 2 3
        // 4 5 6
        // 7 8 9
    }
}
```

## 4. Jagged Array (Different column sizes per row)

```java
class Hello {
    public static void main(String args[]) {

        int jagged[][] = new int[3][];     // 3 rows, columns not fixed
        jagged[0] = new int[]{1, 2};       // row 0 has 2 columns
        jagged[1] = new int[]{3, 4, 5};    // row 1 has 3 columns
        jagged[2] = new int[]{6};          // row 2 has 1 column

        for (int i = 0; i < jagged.length; i++)
        {
            for (int j = 0; j < jagged[i].length; j++)
            {
                System.out.print(jagged[i][j] + " ");
            }
            System.out.println();
        }
        // Output:
        // 1 2
        // 3 4 5
        // 6
    }
}
```

## 5. Array of Objects

```java
class Student {
    String name;
    int marks;

    Student(String name, int marks) {
        this.name = name;
        this.marks = marks;
    }
}

class Hello {
    public static void main(String args[]) {

        Student students[] = new Student[3];
        students[0] = new Student("Ashish", 90);
        students[1] = new Student("Rahul", 85);
        students[2] = new Student("Priya", 95);

        for (Student s : students)
        {
            System.out.println(s.name + " : " + s.marks);
        }
    }
}
```

## Interview Tricky Points
- Array default values: `int[] = 0`, `boolean[] = false`, `String[] = null`
- `ArrayIndexOutOfBoundsException` if you access wrong index
- Arrays are **objects** in Java (stored in heap memory)
- Array size CANNOT change. Need dynamic size? Use `ArrayList` (topic 31)
