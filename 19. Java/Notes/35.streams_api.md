# 35. Streams API (Java 8+)

- Stream = process **collections** (list, set) in a clean, functional way
- Like a pipeline: data flows through operations one by one
- Does NOT modify the original collection

## 1. Basic Stream Operations

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

class Hello {
    public static void main(String args[]) {

        List<Integer> nums = Arrays.asList(5, 3, 8, 1, 9, 2, 7, 4, 6);

        // FILTER - keep elements that match condition
        List<Integer> evenNums = nums.stream()
            .filter(n -> n % 2 == 0)
            .collect(Collectors.toList());
        System.out.println("Even: " + evenNums); // [8, 2, 4, 6]

        // MAP - transform each element
        List<Integer> doubled = nums.stream()
            .map(n -> n * 2)
            .collect(Collectors.toList());
        System.out.println("Doubled: " + doubled); // [10, 6, 16, 2, 18, 4, 14, 8, 12]

        // SORTED
        List<Integer> sorted = nums.stream()
            .sorted()
            .collect(Collectors.toList());
        System.out.println("Sorted: " + sorted); // [1, 2, 3, 4, 5, 6, 7, 8, 9]

        // FOREACH - just do something with each element
        nums.stream().forEach(n -> System.out.print(n + " ")); // 5 3 8 1 9 2 7 4 6
        System.out.println();

        // COUNT
        long count = nums.stream().filter(n -> n > 5).count();
        System.out.println("Count > 5: " + count); // 3

        // MIN / MAX
        int min = nums.stream().min(Integer::compareTo).get();
        int max = nums.stream().max(Integer::compareTo).get();
        System.out.println("Min: " + min + ", Max: " + max); // Min: 1, Max: 9

        // SUM
        int sum = nums.stream().mapToInt(Integer::intValue).sum();
        System.out.println("Sum: " + sum); // 45
    }
}
```

## 2. Chaining (Pipeline)

```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

class Hello {
    public static void main(String args[]) {

        List<String> names = Arrays.asList("ashish", "rahul", "priya", "amit", "ankit");

        // Chain multiple operations together
        List<String> result = names.stream()
            .filter(name -> name.startsWith("a"))   // keep names starting with "a"
            .map(name -> name.toUpperCase())          // convert to uppercase
            .sorted()                                  // sort alphabetically
            .collect(Collectors.toList());              // collect to list

        System.out.println(result); // [AMIT, ANKIT, ASHISH]
    }
}
```

## 3. Stream with Objects

```java
import java.util.ArrayList;
import java.util.List;
import java.util.stream.Collectors;

class Student {
    String name;
    int marks;
    Student(String name, int marks) {
        this.name = name;
        this.marks = marks;
    }
}

class Hello {
    public static void main(String args[]) {

        List<Student> students = new ArrayList<>();
        students.add(new Student("Ashish", 90));
        students.add(new Student("Rahul", 45));
        students.add(new Student("Priya", 95));
        students.add(new Student("Amit", 35));

        // Get names of students who passed (marks >= 50)
        List<String> passedNames = students.stream()
            .filter(s -> s.marks >= 50)
            .map(s -> s.name)
            .collect(Collectors.toList());

        System.out.println("Passed: " + passedNames); // [Ashish, Priya]

        // Average marks
        double avg = students.stream()
            .mapToInt(s -> s.marks)
            .average()
            .getAsDouble();
        System.out.println("Average: " + avg); // 66.25
    }
}
```

## Common Stream Methods
| Method      | What it does                    | Returns      |
|-------------|----------------------------------|--------------|
| `filter()`  | Keep matching elements          | Stream       |
| `map()`     | Transform each element          | Stream       |
| `sorted()`  | Sort elements                   | Stream       |
| `forEach()` | Do action on each element       | void         |
| `collect()` | Convert stream to List/Set/Map  | Collection   |
| `count()`   | Count elements                  | long         |
| `min()`     | Find minimum                    | Optional     |
| `max()`     | Find maximum                    | Optional     |
| `reduce()`  | Combine all into single value   | Optional     |
| `distinct()`| Remove duplicates               | Stream       |
