# 21. Type Casting with Objects (Upcasting & Downcasting)

## 1. Upcasting (Child to Parent) - Automatic

```java
class Animal {
    void sound() {
        System.out.println("Animal sound");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("Bark!");
    }

    void fetch() {
        System.out.println("Dog fetches ball");
    }
}

class Hello {
    public static void main(String args[]) {

        // UPCASTING: Child object stored in Parent reference
        Animal a = new Dog(); // Dog is "upcasted" to Animal -- AUTOMATIC
        a.sound();   // Bark! (Dog's method - runtime polymorphism)
        // a.fetch(); // ERROR! Animal reference can't see Dog-specific methods
    }
}
```

## 2. Downcasting (Parent to Child) - Manual

```java
class Animal {
    void sound() {
        System.out.println("Animal sound");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("Bark!");
    }

    void fetch() {
        System.out.println("Dog fetches ball");
    }
}

class Hello {
    public static void main(String args[]) {

        Animal a = new Dog(); // upcasted
        // a.fetch();         // ERROR! Can't call Dog methods through Animal ref

        // DOWNCASTING: cast it back to Dog to access Dog methods
        Dog d = (Dog) a;      // Manual cast - works because actual object IS a Dog
        d.fetch();             // Dog fetches ball -- NOW it works!

        // DANGER: Wrong downcasting throws ClassCastException
        Animal a2 = new Animal();
        // Dog d2 = (Dog) a2; // RUNTIME ERROR! a2 is NOT a Dog

        // SAFE WAY: Check with instanceof before casting
        if (a2 instanceof Dog) {
            Dog d2 = (Dog) a2;
            d2.fetch();
        } else {
            System.out.println("a2 is not a Dog!");
        }
    }
}
```

## 3. instanceof Operator

```java
class Animal { }
class Dog extends Animal { }
class Cat extends Animal { }

class Hello {
    public static void main(String args[]) {

        Animal a = new Dog();

        System.out.println(a instanceof Dog);    // true
        System.out.println(a instanceof Animal); // true (Dog IS-A Animal)
        System.out.println(a instanceof Cat);    // false

        // Useful in real code:
        Animal animals[] = { new Dog(), new Cat(), new Dog() };
        for (Animal animal : animals) {
            if (animal instanceof Dog) {
                System.out.println("Found a Dog!");
            } else if (animal instanceof Cat) {
                System.out.println("Found a Cat!");
            }
        }
    }
}
```

## Interview Points
- Upcasting = automatic, safe, always works
- Downcasting = manual, risky, can throw `ClassCastException`
- Always use `instanceof` before downcasting
- Upcasting is the basis of **runtime polymorphism**
