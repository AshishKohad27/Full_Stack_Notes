# 2. Print Statements

## 3 Ways to Print

```java
class Hello {
    public static void main(String args[]) {

        // 1. println - prints and goes to NEXT line
        System.out.println("Hello");
        System.out.println("World");
        // Output:
        // Hello
        // World

        // 2. print - prints and stays on SAME line
        System.out.print("Hello ");
        System.out.print("World");
        // Output: Hello World

        // 3. printf - formatted print (like C language)
        String name = "Ashish";
        int age = 22;
        System.out.printf("My name is %s and age is %d", name, age);
        // Output: My name is Ashish and age is 22
    }
}
```

## printf Format Specifiers

| Symbol | Used For        | Example          |
|--------|-----------------|------------------|
| `%s`   | String          | "Ashish"         |
| `%d`   | Integer         | 42               |
| `%f`   | Float/Double    | 3.14             |
| `%.2f` | 2 decimal places| 3.14             |
| `%c`   | Character       | 'A'              |
| `%b`   | Boolean         | true             |
| `%n`   | New line        |                  |

## Concatenation (Joining)

```java
class Hello {
    public static void main(String args[]) {

        String name = "Ashish";
        int age = 22;

        // Using + to join strings
        System.out.println("Name: " + name + ", Age: " + age);
        // Output: Name: Ashish, Age: 22
    }
}
```
