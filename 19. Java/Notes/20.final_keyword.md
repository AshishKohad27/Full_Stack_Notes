# 20. Final Keyword

- `final` = **cannot be changed**
- Works on: variables, methods, classes

## 1. final Variable (Constant)

```java
class Hello {
    public static void main(String args[]) {

        final int MAX = 100;
        // MAX = 200; // ERROR! Cannot change final variable

        final String NAME = "Ashish";
        // NAME = "Rahul"; // ERROR!

        System.out.println(MAX);  // 100
        System.out.println(NAME); // Ashish

        // Convention: final variable names in UPPERCASE
        final double PI = 3.14159;
        final int MAX_SIZE = 50;
    }
}
```

## 2. final Method (Cannot Override)

```java
class Parent {
    final void show() {
        System.out.println("Parent's show - cannot override me!");
    }
}

class Child extends Parent {
    // void show() { } // ERROR! Cannot override final method
}

class Hello {
    public static void main(String args[]) {

        Child c = new Child();
        c.show(); // Parent's show - cannot override me!
    }
}
```

## 3. final Class (Cannot Extend / No Inheritance)

```java
final class Secure {
    void display() {
        System.out.println("I am final class - no one can extend me");
    }
}

// class Hacker extends Secure { } // ERROR! Cannot extend final class

class Hello {
    public static void main(String args[]) {

        Secure s = new Secure();
        s.display(); // I am final class - no one can extend me
    }
}
```

## Interview Quick Reference
| Usage           | Meaning                                |
|-----------------|----------------------------------------|
| `final variable`| Value cannot change (constant)         |
| `final method`  | Cannot be overridden by child          |
| `final class`   | Cannot be extended (no child classes)  |

## Interview Points
- `String` class is `final` - that's why you can't extend String!
- `final` variable MUST be initialized (at declaration or in constructor)
- `final` does NOT mean immutable for objects (you can change object's properties, just can't reassign)

```java
class Hello {
    public static void main(String args[]) {

        final int[] arr = {1, 2, 3};
        arr[0] = 99;         // OK! Changing content is allowed
        // arr = new int[5]; // ERROR! Cannot reassign the reference
    }
}
```
