# 33. Enum (Enumeration)

- Enum = a set of **fixed constants** (values that never change)
- Like a dropdown menu - only predefined options

## 1. Basic Enum

```java
// Define enum OUTSIDE class (or inside class)
enum Day {
    MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, SUNDAY
}

class Hello {
    public static void main(String args[]) {

        Day today = Day.WEDNESDAY;
        System.out.println(today); // WEDNESDAY

        // Use in switch
        switch (today) {
            case MONDAY:
            case TUESDAY:
            case WEDNESDAY:
            case THURSDAY:
            case FRIDAY:
                System.out.println("Weekday");
                break;
            case SATURDAY:
            case SUNDAY:
                System.out.println("Weekend!");
                break;
        }
        // Output: Weekday
    }
}
```

## 2. Enum with Values and Constructor

```java
enum Status {
    SUCCESS(200, "OK"),
    NOT_FOUND(404, "Not Found"),
    ERROR(500, "Server Error");

    int code;
    String message;

    // Enum constructor (always private)
    Status(int code, String message) {
        this.code = code;
        this.message = message;
    }
}

class Hello {
    public static void main(String args[]) {

        Status s = Status.NOT_FOUND;
        System.out.println(s);          // NOT_FOUND
        System.out.println(s.code);     // 404
        System.out.println(s.message);  // Not Found

        // Loop through all enum values
        for (Status status : Status.values()) {
            System.out.println(status.code + " - " + status.message);
        }
        // 200 - OK
        // 404 - Not Found
        // 500 - Server Error
    }
}
```

## 3. Useful Enum Methods

```java
enum Color { RED, GREEN, BLUE }

class Hello {
    public static void main(String args[]) {

        // values() - returns array of all constants
        Color[] colors = Color.values();

        // valueOf() - string to enum
        Color c = Color.valueOf("RED");
        System.out.println(c); // RED

        // ordinal() - position (0-based)
        System.out.println(Color.RED.ordinal());   // 0
        System.out.println(Color.GREEN.ordinal()); // 1
        System.out.println(Color.BLUE.ordinal());  // 2

        // name() - returns name as string
        System.out.println(Color.RED.name()); // RED
    }
}
```

## Interview Points
- Enum is **type-safe** (can only use predefined values)
- Enum constructor is always **private**
- Enum can have fields, methods, and constructors
- Enum values are `public static final` by default
- Enum is better than `int` constants: `Status.SUCCESS` vs `200`
- Enum can implement interfaces but cannot extend classes
