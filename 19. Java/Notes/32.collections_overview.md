# 32. Collections Framework - Overview

- Collections = **ready-made data structures** in Java
- All in `java.util` package

## Collections Hierarchy
```
                    Collection (Interface)
                    /          |          \
                 List        Set         Queue
                /    \       /   \          |
         ArrayList  LinkedList  HashSet  PriorityQueue
                               TreeSet
                               LinkedHashSet

                    Map (Interface) -- separate from Collection
                   /       |        \
              HashMap  TreeMap   LinkedHashMap
```

## Quick Comparison

| Collection     | Ordered | Duplicates | Null | When to Use                       |
|----------------|---------|------------|------|-----------------------------------|
| `ArrayList`    | Yes     | Yes        | Yes  | Most common, fast read by index   |
| `LinkedList`   | Yes     | Yes        | Yes  | Fast insert/delete in middle      |
| `HashSet`      | No      | No         | 1    | Unique elements, fast lookup      |
| `TreeSet`      | Sorted  | No         | No   | Unique + sorted                   |
| `HashMap`      | No      | Values yes | Yes  | Key-value pairs, fast lookup      |
| `TreeMap`      | Sorted  | Values yes | No   | Key-value + sorted by key         |

## 1. HashSet (Unique elements, no duplicates)

```java
import java.util.HashSet;

class Hello {
    public static void main(String args[]) {

        HashSet<String> fruits = new HashSet<>();
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Apple");   // duplicate - IGNORED!
        fruits.add("Mango");

        System.out.println(fruits);          // [Apple, Banana, Mango] (no duplicate!)
        System.out.println(fruits.size());   // 3
        System.out.println(fruits.contains("Apple")); // true

        fruits.remove("Banana");
        System.out.println(fruits);          // [Apple, Mango]
    }
}
```

## 2. LinkedList

```java
import java.util.LinkedList;

class Hello {
    public static void main(String args[]) {

        LinkedList<String> list = new LinkedList<>();
        list.add("A");
        list.add("B");
        list.add("C");

        list.addFirst("Start");  // add at beginning
        list.addLast("End");     // add at end

        System.out.println(list);         // [Start, A, B, C, End]
        System.out.println(list.getFirst()); // Start
        System.out.println(list.getLast());   // End

        list.removeFirst();
        list.removeLast();
        System.out.println(list); // [A, B, C]
    }
}
```

## 3. TreeSet (Sorted, no duplicates)

```java
import java.util.TreeSet;

class Hello {
    public static void main(String args[]) {

        TreeSet<Integer> nums = new TreeSet<>();
        nums.add(40);
        nums.add(10);
        nums.add(30);
        nums.add(20);
        nums.add(10); // duplicate ignored

        System.out.println(nums); // [10, 20, 30, 40] -- automatically sorted!
    }
}
```

## 4. Iterator - Safe way to loop and remove

```java
import java.util.ArrayList;
import java.util.Iterator;

class Hello {
    public static void main(String args[]) {

        ArrayList<Integer> nums = new ArrayList<>();
        nums.add(10);
        nums.add(20);
        nums.add(30);
        nums.add(40);

        Iterator<Integer> it = nums.iterator();
        while (it.hasNext()) {
            int num = it.next();
            if (num == 20) {
                it.remove(); // safe removal during iteration
            }
        }

        System.out.println(nums); // [10, 30, 40]
    }
}
```

## Interview Points
- `List` = ordered, allows duplicates
- `Set` = unordered, NO duplicates
- `Map` = key-value pairs
- `ArrayList` vs `LinkedList`: ArrayList is faster for reading, LinkedList faster for insert/delete
- `HashSet` uses `hashCode()` + `equals()` to detect duplicates
